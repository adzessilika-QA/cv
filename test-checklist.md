# ✅ Пример тестового чек-листа

## Чек-лист для формы регистрации

### Основной сценарий
- [ ] Регистрация с валидными данными
- [ ] Подтверждение email (письмо приходит)
- [ ] Автоматический вход после регистрации

### Валидация полей
- [ ] Email: только правильный формат
- [ ] Пароль: минимальная длина 8 символов
- [ ] Пароль и подтверждение: должны совпадать
- [ ] Обязательные поля: помечены звездочкой

### Ошибки и сообщения
- [ ] Email уже занят → понятное сообщение
- [ ] Слабый пароль → подсказка
- [ ] Пустые обязательные поля → подсветка
- [ ] Несовпадающие пароли → ошибка

## Чек-лист для главной страницы сайта

### Загрузка и отображение
- [ ] Страница грузится за <3 секунды
- [ ] Все изображения загружаются
- [ ] Шрифты отображаются правильно
- [ ] Нет ошибок в консоли браузера

### Навигация
- [ ] Логотип ведет на главную
- [ ] Все пункты меню работают
- [ ] Поиск выдает результаты
- [ ] Хлебные крошки (breadcrumbs) правильные

### Адаптивность
- [ ] На мобильном телефоне
- [ ] На планшете
- [ ] На десктопе
- [ ] Горизонтальный и вертикальный экран
