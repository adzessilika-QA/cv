# Пример баг-репорта

**Заголовок:** Кнопка "Отправить" не работает в форме обратной связи

**Шаги воспроизведения:**
1. Открыть сайт example.com/contact
2. Заполнить все поля формы
3. Нажать кнопку "Отправить"

**Ожидаемый результат:** Форма отправляется
**Фактический результат:** Кнопка не реагирует

**Серьезность:** High
**Приоритет:** High
**Окружение:** Chrome 121, Windows 11

**Дополнительно:**
- В консоли ошибка: "submitForm is not defined"
- Проблема в 100% случаев
